# pdf-page-counter-bat

## Description  
Output the number of pages of PDF to csv.  

## Similar tool  
https://github.com/KentaGoto/pdf_page_counter

## Usage  
1. Double-click to run.  
2. Drag and drop the target folder to the command prompt and enter.  

## Requirements  
- Windows  
- grep  
- sed  
- pdfinfo
